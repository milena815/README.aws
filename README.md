RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 12/05/2026
Empresa: Abstergo Industries
Responsável: Milena Barbosa

Introdução:
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Milena, com o objetivo de elencar três serviços AWS para reduzir custos operacionais e aumentar a eficiência dos processos internos.

Descrição do Projeto:
O projeto foi dividido em três etapas, cada uma com objetivos específicos:

Etapa 1
Nome da ferramenta: Amazon EC2

Foco da ferramenta: Hospedagem e processamento de aplicações

Descrição do caso de uso: Hospedagem de aplicações com escalabilidade automática, garantindo alta disponibilidade e menor custo de infraestrutura.

Etapa 2
Nome da ferramenta: Amazon S3

Foco da ferramenta: Armazenamento de dados e arquivos

Descrição do caso de uso: Armazenamento seguro e durável de relatórios, backups e arquivos, substituindo servidores locais.


Etapa 3
Nome da ferramenta: Amazon RDS

Foco da ferramenta: Banco de dados relacional gerenciado

Descrição do caso de uso: Banco de dados relacional gerenciado, com manutenção automática, redundância e maior confiabilidade.

Conclusão:
A implementação das ferramentas AWS na empresa Abstergo Industries trouxe benefícios como redução de custos, melhoria na performance e aumento da produtividade. Recomenda-se a continuidade do uso das soluções implementadas e a busca por novas tecnologias que possam otimizar ainda mais os processos da empresa.
